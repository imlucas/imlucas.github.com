imlucas.github.io/mott

mott ties together tools to help you make apps. it makes using great new tools like browserify, less, and phonegap simple. mott is a companion to Make, not a replacement. it provides tested recipes for deploying single page JS apps, documentation sites, complex mobile apps, or just mockups. code and configuration are completely reusable.

imlucas.github.io/node-mambo

A document mapper that provides a clean, fluent api for AWS DynamoDB. mambo provides all of the builtin power of dynamo with higher level abstractions to handle key value casting, smart update operations, and powerful parallelization.

github.com/exfm/node-magneto

Unit testing DynamoDB backed services can be a real pain. You should be able to run your tests from anywhere and it should be much faster than making REST calls to Virginia. magneto is an implementation of the dynamo API in node.js that can easily be used to test your node.js apps locally or in ci, quickly without having to pay for extra provisioning just to do your work.
